# ListaExerciciosLogicaBasica_II
Lista de Exercicios de Logica Basica em C - PARTE 2

## Declaração de Variáveis
1. Crie um programa que declare uma variável double e outra int, atribua valores
a elas e calcule a soma, exibindo o resultado.
2. Crie um programa que declare uma variável char e leia uma letra do usuário,
depois exiba a letra digitada junto com uma mensagem personalizada.
## Estruturas Condicionais (if / else)
3. Leia três números inteiros e mostre qual é o maior e qual é o menor.
4. Leia a idade de uma pessoa e informe se ela é menor de idade, adulto ou idoso
(idade ≥ 60).
5. Peça ao usuário para digitar um número e informe se ele é positivo, negativo ou
zero.
## Laços de Repetição (for / while)
6. Mostre os números pares de 2 a 50 usando for.
7. Leia um número inteiro e mostre sua tabuada de 1 a 12.
8. Calcule a soma dos números pares de 1 a 200 usando while.
9. Leia um número N e calcule a soma dos N primeiros números ímpares usando
for.
10. Leia números do usuário até que a soma deles ultrapasse 100, usando while.
## Estrutura Switch
11. Crie um menu com as opções: Quadrado, Cubo e Raiz Quadrada. Peça um
número e execute a operação correspondente.
12. Peça um número de 1 a 12 e exiba a estação do ano correspondente
(considerando hemisfério sul).
13. Crie um programa que receba um código de cor (1-vermelho, 2-verde, 3-azul, 4-
amarelo) e exiba o nome da cor.
## Desafios (Misturando Conceitos)
14. Crie um programa que leia o peso e altura de 5 pessoas, calcule o IMC de cada
uma e exiba se está abaixo, no peso ideal ou acima.
15. Leia um número inteiro positivo e informe se ele é um número perfeito (a soma
de seus divisores próprios é igual ao número).
16. Crie um programa que simule um sistema de votação: o usuário digita o voto (1-
candidato A, 2-candidato B, 3-branco) e o programa conta votos até o usuário
digitar 0. Exiba o total de cada opção.
17. Desenvolva um programa que leia números inteiros e determine o maior número
digitado, encerrando quando o usuário digitar um número negativo.
